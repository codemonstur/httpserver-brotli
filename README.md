# HTTP server Brotli compression support

Adds Brotli compression support to the httpserver.

## Usage

Add the maven dependency:

    <dependency>
        <groupId>codemonstur</groupId>
        <artifactId>httpserver-brotli</artifactId>
        <version>0.1.0</version>
    </dependency>
